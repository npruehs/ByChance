Anchor3D
Placeholder for game elements in 3D levels that can be filled after the level generation process.
BoxF
Box with floating point position and extents.
Chunk3D

Main building block that makes up a 3D level, providing extents, position and rotation.

Chunks are created based on pre-defined chunk templates and define the positions of all contexts and anchors associated with the chunk, as well as attributes that are required for the level generation process such as the probability of a chunk being picked to be added next.

ChunkLibrary3D
Collection of user-defined 2D chunk templates.
ChunkTemplate3D
Template that is used for creating similar chunks, which in turn make up a 3D level. Chunk templates define the chunk extents, the positions of all contexts and anchors of chunks, as well as attributes that are required for the level generation process such as the probability of a chunk being picked to be added next.
Context3D
Describes a position a chunk can be aligned at to another one in a 3D level.
Level3D
3D level of a given width, height and depth that consists of a number of chunks.
LevelGenerator3D
Generates a 3D level based on a given chunk library.
