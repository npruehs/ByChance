ChunkTemplate2D(Vector2F)
Constructs a new chunk template for chunks with the specified width and height and a default weight. Chunks constructed with this template may not be rotated by the level generator.
ChunkTemplate2D(Vector2F, Boolean)
Constructs a new chunk template for chunks with the specified width, height and rotation permission. Chunks constructed with this template will have a default weight and may not be rotated by the level generator.
ChunkTemplate2D(Vector2F, Int32)
Constructs a new chunk template for chunks with the specified width and height and the passed weight. Chunks constructed with this template may not be rotated by the level generator.
ChunkTemplate2D(Vector2F, String)
Constructs a new chunk template for chunks with the specified width and height and a default weight. Chunks constructed with the new template will be of of the passed category and may not be rotated by the level generator.
ChunkTemplate2D(Vector2F, Int32, String, Boolean)
Constructs a new chunk template for chunks with the specified width, height, weight, category and rotation permission.
